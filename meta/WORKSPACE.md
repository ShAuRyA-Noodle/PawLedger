## How to resume from this state

```bash
git clone https://github.com/ShAuRyA-Noodle/Mountain-Mommy.git pawledger
cd pawledger
pnpm install
cp .env.example .env       # fill in values per docs/LAUNCH_CHECKLIST.md
pnpm db:push
pnpm db:seed
pnpm dev
```

Demo admin: `admin@pawledger.org` (magic link prints to console in dev when RESEND not configured).
